Sitemap

The improved well-structured layout for the new COMP 1950 site provides easy navigation through the course content. All course resources are organized and easy to reference in multiple ways:

  • home page dashboard
  • scrolling navigation side panel
  • search field

Architecture and Content Overview

The top level architecture consists of a home page dashboard with six links to the main pages of the site:

  • About
  • Lectures
  • Exercises
  • Quizzes
  • Assignments
  • Resources

About

This page contains overall course information:

  • course description
  • course outline/curriculum (including timetable, room information, grading, and BCIT policies)
  • instructor contact information

The contact information for various instructors is stored in external server side include files so that the information for the appropriate instructor can be easily updated on the site by simplying modifying the SSI include statement.

Lectures

The lectures page will introduce the basic layout and outline the lesson plan. Links will be provided to separate web page for each lesson as follows:

  • lesson 1 – tools and standards
  • lesson 2 – html5
  • lesson 3 – css techniques
  • lesson 4 – css layouts
  • lesson 5 – responsive css
  • lesson 6 – css3
  • lesson 7 – javascript frameworks
  • lesson 8 – dynamic content
  • lesson 9 – templating, seo
  • lesson 10 – methodology

The content for each lesson will be stored in external server side files where they can be easily updated, if necessary, and will automatically update on the site through the SSI include statement. Each lesson will only be available when "enabled" by the instructor so that students will not work ahead of the scheduled curriculum.

Each lesson page is organized with agenda, curriculum, exercises, and assignments sections. The scrolling navigation side panel will link to each section.

Exercises

The main Exercises page will provide an outline of the exercises to be covered throughout the course, and any tips such as developer tools. Links will be provided to separate web page for each exercise for each lesson. Each exercise page is also accessible through the lesson page content.

Each exercise will only be available when "enabled" by the instructor so that students will not work ahead of the scheduled curriculum.

Quizzes

The quizzes page will provide instructions on the use of the online quiz system. A grade distribution chart with class average for each quiz may also be made availabe to students on this page so that they can gauge their progress.

Links to each online quiz will be provided. Each quiz link will only be available when "enabled" by the instructor at the start of the quiz session.

Assignments

The assignments main page will provide instructions on how and when to submit/upload assignments. This will include the step by step instructions for accessing the bcitcomp.ca server.

There will also be links to each assignment that will only be available once "enabled" by the instructor so that students will not work ahead of the scheduled curriculum.

Resources

This page will contain all relevant links to resources that the students may find useful. This would include links to all officially related sources:

  • BCIT Home Page
  • MyBCIT
  • COMP 1950 Course Details Page on BCIT.ca

and links to web development and design tools, tips, best practices, validators, APIs, etc.
